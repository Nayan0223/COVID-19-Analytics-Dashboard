# 🦠 COVID-19 Analytics Dashboard

## Overview
This project presents an interactive Tableau dashboard for analyzing COVID-19 statistics across different countries. The dashboard provides insights into total cases, deaths, population metrics, and economic indicators.

## Objectives
- Monitor COVID-19 trends across countries.
- Compare total cases and deaths.
- Analyze population and GDP-related metrics.
- Create interactive visualizations for decision-making.

## Tools & Technologies
- Tableau
- Microsoft Excel
- Data Cleaning
- Data Visualization

## Dataset
COVID-19 Global Dataset

## Dashboard KPIs
- Total Population
- Total Cases
- Total Deaths

## Visualizations
- Top Cases by Country
- Top Deaths by Country
- High GDP Countries Analysis
- KPI Summary Cards

## Key Insights
- India and Brazil reported the highest case counts in the selected dataset.
- Country-wise death comparisons reveal significant variations.
- GDP and population indicators help provide additional context for COVID-19 impact analysis.
